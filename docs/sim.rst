**************************************
Simulation
**************************************

The package provides functions for simulations related IVIM,
specifically

* Simulation of noisy data given IVIM parameters (:func:`API <ivim.sim.noise>`)
* Particle velocities according to the Langevin equation (:func:`API <ivim.sim.langevin>`)
